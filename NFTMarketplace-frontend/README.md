# Readme

## 一、项目说明

### 功能说明
1. 连接钱包并获取后端 JWT token
2. 上架 NFT
3. 取消上架 NFT
4. 显示上架 NFT 详情（ID、价格等） 
5. 购买 NFT

### 替换合约地址
替换合约地址，复制合约地址到 config.ts 中的 MARKETPLACE_ADDRESS 值

### 注册 Project ID

在网站 https://dashboard.walletconnect.com/ 注册 project ID，将 ID 值复制到 wagmi.ts 中的 WALLET_CONNECT_PROJECT_ID 值

### 安装依赖
```bash
# 环境安装
npm install
```

### 启动运行

```bash
# 运行
npm run dev
```
